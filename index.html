Terminal Refresher:

use the terminal to create a "projects" folder - or whatever you'd like to call it - using the command:

$ mkdir projects (make directory)

You dont need to type the "$" in terminal, this is just how terminal commands are denoted

change your current directory to the projects folder/directory using:

$ cd projects (change directory)

created an index.html file using:

$ touch index.html (create file)

Attempt to open the projects folder in VScode using:

$ code . (open all in VSCode)

If using the "code ." command gives an error of open VSCode and use the fuzzy finder:

cmd + shift + p and search for "shell"


click "install code command to PATH"

open a new terminal window, navigate back to your project directory using the cd command from step 2 and try the command

$ code .

hopefully you opened up an index.html file in VScode!
